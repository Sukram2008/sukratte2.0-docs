# 🕵️‍♂️✅ | Identität entsperren

Befehl: **`/moderate-anonymous-channel enable`**

Optionen:
- **`display-name:`**  
  Gebe die Identität ein, welche entsperrt werden soll.

<u>Beschreibung:</u>  
 Entsperre eine Identität, also einen NUtzer wieder vom Anonymen Chat System.
