# 🕵️‍♂️✍ | Anonyme Nachricht löschen

Befehl: **`/moderate-anonymous-channel delete message`**

Optionen:
- **`message:`**  
  Wähle dafür die Nachricht aus dem Drop-Down aus oder gebe die Nachrichten-ID ein.

<u>Beschreibung:</u>  
 Es kann eine Nachricht, welche im  Anonymous Chat gesendet wurde, gelöscht werden.
