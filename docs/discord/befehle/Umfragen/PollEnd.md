# 📊🛑 | Umfragen: end

Befehl: **`/poll end`**

### Erforderliche Optionen
- **`msg-id:`**
  Die ID der Nachricht, welche die Umfrage enthält.

:::tip Hinweis
Beim Tippen der Option **`msg-id`** wird dir automatisch eine Liste aller aktuell laufenden Umfragen angezeigt, aus der du die richtige auswählen kannst.
:::

### Beschreibung
Beendet eine aktive Umfrage sofort. Nach dem Ausführen können keine weiteren Stimmen mehr abgegeben werden und das Endergebnis wird fixiert.
