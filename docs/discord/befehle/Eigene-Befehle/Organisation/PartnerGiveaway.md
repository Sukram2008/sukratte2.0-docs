# 🤝🎁 | Partner Giveaway

Befehl: **`/partner-giveaway`**

### Erforderliche Optionen
- **`partner`**
  Name oder Kontaktperson unseres Partners

### Beschreibung
Informiert den Partner über ein Gewinnspiel vor der offiziellen Ankündigung.

### Berechtigungen
