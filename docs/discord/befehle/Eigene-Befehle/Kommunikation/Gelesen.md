# 📖✅ | Nachricht gelesen

Befehl: **`/gelesen`**

### Erforderliche Optionen
- **`hinweis`**
  Kurze Angabe, welche Nachricht oder Anweisung bestätigt wird

### Beschreibung
Wird vom Team genutzt, um den Erhalt und das Verständnis von wichtigen internen Informationen offiziell zu quittieren.
