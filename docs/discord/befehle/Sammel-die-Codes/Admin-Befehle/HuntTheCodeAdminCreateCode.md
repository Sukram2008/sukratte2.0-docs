# 🆕🔑 | Code erstellen

Befehl: **`/hunt-the-code-admin create-code`**

### Erforderliche Optionen
- **`code:`**
  Der Text des Codes, der gefunden werden muss.
- **`points:`**
  Die Anzahl der Punkte, die das Einlösen einbringt.

### Beschreibung
Mit diesem Befehl generierst du neue Codes für das Event. Du kannst festlegen, wie viele Punkte ein Code wert ist.
