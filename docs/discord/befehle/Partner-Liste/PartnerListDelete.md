# 🤝🗑️ | Partner löschen

Befehl: **`/partner delete`**

### Erforderliche Optionen
- **`id:`**
  Die eindeutige ID des Partners, der gelöscht werden soll.

### Beschreibung
Entfernt den angegebenen Partner dauerhaft aus der Datenbank.

### Berechtigungen
