# 🛑🎁 | Gewinnspiel beenden

Befehl: **`/gmanage end`**

### Erforderliche Optionen
- **`msg-id:`**
  Die ID der Gewinnspiel-Nachricht.

### Beschreibung
Beendet ein aktuell laufendes Gewinnspiel sofort und lost die Gewinner aus.

### Berechtigungen
