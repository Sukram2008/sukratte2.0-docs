# 🎲🎁 | Gewinnspiel neu auslosen

Befehl: **`/gmanage reroll`**

### Erforderliche Optionen
- **`msg-id:`**
  Die ID der Gewinnspiel-Nachricht.

### Andere Optionen
- **`winner-count:`** [Optional]
  Anzahl der neuen Gewinner, die gezogen werden sollen.
- **`prevent-duplicates:`** [Optional]
  Legt fest, ob Nutzer nicht doppelt gewinnen können (True/False).

### Beschreibung
Zieht einen oder mehrere neue Gewinner für ein bereits beendetes Gewinnspiel.

### Berechtigungen
