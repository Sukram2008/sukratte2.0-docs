# 🔓✅ | Geburtstag entsperren

Befehl: **`/manage-birthday unlock`**

Optionen:
- **`user:`**
  Der Nutzer, dessen Eintrag entsperrt werden soll.

<u>Beschreibung:</u>  
  Hebt die Sperre eines Geburtstagseintrags auf, sodass der Nutzer diesen wieder selbst bearbeiten kann.

<u>Berechtigungen</u>  
  Administrator
