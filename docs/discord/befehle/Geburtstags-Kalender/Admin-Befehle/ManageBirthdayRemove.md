# 🛠️🗑️ | Nutzer-Eintrag entfernen

Befehl: **`/manage-birthday remove`**

Optionen:
- **`user:`**
  Der Nutzer, dessen Geburtstag entfernt werden soll.

<u>Beschreibung:</u>  
  Entfernt den hinterlegten Geburtstag eines spezifischen Nutzers dauerhaft vom Server.

<u>Berechtigungen</u>  
  Administrator
