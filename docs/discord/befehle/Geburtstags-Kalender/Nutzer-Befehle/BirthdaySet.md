# 🎂📅 | Geburtstag eintragen
Befehl: **`/birthday set`**

### Erforderliche Optionen
Optionen:
- **`day:`**
Der Tag deiner Geburt (1-31)
- **`month:`**
Der Monat deiner Geburt (1-12)

### Andere Optionen
- **`year:`** [Optional]  
  Jahr deines Geburtstags.

### Beschreibung
Mit diesem Befehl trägst du dich in den serverweiten Geburtstagskalender ein. 
- An deinem Ehrentag wird eine automatische Nachricht im dafür vorgesehenen Kanal gesendet.
- Du erhältst (je nach Server-Einstellung) eine spezielle Geburtstagsrolle für diesen Tag.
