# 💬🚫 | Nachrichten-Belohnungen deaktivieren

Befehl: **`/economy msg_drop_msg disable`**

Optionen:
Keine Optionen vorhanden.

<u>Beschreibung:</u>  
  Deaktiviere diesen Modus, um das Verdienen von Coins durch das Senden von Nachrichten im Chat zu stoppen.

<u>Berechtigungen</u>  
  Administratoren
