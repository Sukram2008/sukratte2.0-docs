# 🚨💸 | Raub begehen

Befehl: **`/economy rob`**

Optionen:
- **`user:`**
  Der Nutzer welchem Geld von der Balance geklaut werden soll.

<u>Beschreibung:</u>  
  Versuche, einem anderen Nutzer Geld aus seinem Portemonnaie (Balance) zu stehlen. Da das Feld keine Beschreibung darüber hat, ist es eine Pflichtangabe.

<u>Berechtigungen</u>  
  Alle Mitglieder
