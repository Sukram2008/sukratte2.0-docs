# 💰💳 | Kontostand anzeigen

Befehl: **`/economy balance`**

Optionen:
- **`user:`** [Optional]  
  Trage den Nutzer ein von welchem du den Kontostand wissen möchtest.

<u>Beschreibung:</u>  
  Zeige dir von dir oder einem Nutzer den Kontostand vom Economy System an.

<u>Berechtigungen</u>  
  Alle Mitglieder
