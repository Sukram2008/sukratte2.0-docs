# 🧹 | Nachrichten löschen

Befehl: **`/moderate clear`**

### Erforderliche Optionen
- **`amount:`**
  Die Anzahl der zu löschenden Nachrichten.

### Beschreibung
Bereinigt den aktuellen Kanal um die angegebene Anzahl an Nachrichten.

### Berechtigungen
