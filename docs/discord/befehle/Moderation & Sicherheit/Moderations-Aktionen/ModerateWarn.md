# ⚠️ | Nutzer warnen

Befehl: **`/moderate warn`**

### Erforderliche Optionen
- **`user:`**
  Der Nutzer, der verwarnt werden soll.
- **`reason:`**
  Der Grund für die Verwarnung.

### Andere Optionen
- **`proof:`** [Optional]
  Ein Beweis für die Verwarnung.

### Beschreibung
Erteilt einem Nutzer eine offizielle Verwarnung.

### Berechtigungen
