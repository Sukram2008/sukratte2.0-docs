# 🔍 | Notizen anzeigen

Befehl: **`/moderate notes view`**

### Erforderliche Optionen
- **`user:`**
  Nutzer dem du die Notiz ansehen möchtest.

### Beschreibung
Listet alle hinterlegten Notizen für den ausgewählten Benutzer auf.

### Berechtigungen
