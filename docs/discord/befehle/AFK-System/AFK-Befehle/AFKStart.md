# 💤🏁 | AFK-Sitzung starten

Befehl: **`/afk start`**

### Optionen
- **`reason`** [Optional]
  Erkläre, warum du AFK gehst
- **`auto-end`** [Optional]
  Der Bot wird automatisch dein AFK-Status beenden, sobald du eine Nachricht schreibst (Optionen: `True` / `False`)

### Beschreibung
Mit diesem Befehl kannst du dich auf dem Server als abwesend markieren. 
- Wenn dich andere Nutzer erwähnen, während du AFK bist, informiert der Bot sie darüber.
- Über `auto-end` kannst du festlegen, ob der Status bei deiner nächsten Nachricht sofort verschwinden soll oder ob du ihn manuell beenden möchtest.
