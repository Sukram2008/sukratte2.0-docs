# ⌨️💤 | AFK-Modus aktivieren

Befehl: **`/afk start`**

Optionen:
- **`reason:`** [Optional]  
  Der Grund für deine Abwesenheit.
  - **`auto-end:`** [Optional]  
  Automatisches beenden, bei der nächsten Nachricht.
  Gebe "`True`" für Ja oder "`False`" für Nein ein.

<u>Beschreibung:</u>  
  Setzt deinen Status auf AFK. Wenn dich jemand markiert (auch bei einer Antwort), antwortet der Bot mit deinem hinterlegten Grund. Der Status wird automatisch entfernt, sobald du wieder eine Nachricht schreibst.

<u>Berechtigungen</u>  
  Alle Mitglieder
