# ⌨️🔔 | AFK-Modus beenden

Befehl: **`/afk end`**

Optionen:
Keine Optionen vorhanden.

<u>Beschreibung:</u>  
  Beendet deinen AFK-Status manuell, falls du noch keine Nachricht in einen Kanal geschrieben hast (was den Status normalerweise automatisch aufhebt).

<u>Berechtigungen</u>  
  Alle Mitglieder
