# 🔔✅ | AFK-Sitzung beenden

Befehl: **`/afk end`**

### Beschreibung
Beendet deine aktuelle AFK-Sitzung manuell. Dein Status wird zurückgesetzt und Nutzer erhalten keine Benachrichtigung mehr, dass du abwesend bist, wenn sie dich erwähnen.

### Berechtigungen
