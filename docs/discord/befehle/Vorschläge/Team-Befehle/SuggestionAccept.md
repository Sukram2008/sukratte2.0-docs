# ✅💡 | Vorschlag akzeptieren

Befehl: **`/suggestion accept`**

### Erforderliche Optionen
- **`id:`**
  Die Vorschlags-ID, den du akzeptieren möchtest.
- **`comment:`**
  Erklärung, warum der Vorschlag angenommen wird.

### Beschreibung
Mit diesem Befehl akzeptierst du einen eingereichten Vorschlag. Der Status wird auf `✅ Angenommen` geändert und dein Kommentar wird als Begründung unter dem Vorschlag angezeigt.
