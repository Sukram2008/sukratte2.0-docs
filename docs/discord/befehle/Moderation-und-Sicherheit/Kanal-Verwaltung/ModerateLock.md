# 🔒 | Kanal sperren

Befehl: **`/moderate lock`**

### Erforderliche Optionen
- **`reason:`**
  Der Grund für die Sperrung des Kanals.

### Beschreibung
Sperrt den aktuellen Kanal für die Standard-Rolle.
