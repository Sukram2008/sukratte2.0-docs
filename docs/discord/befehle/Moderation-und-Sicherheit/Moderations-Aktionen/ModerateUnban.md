# 🔓 | Nutzer entbannen

Befehl: **`/moderate unban`**

### Erforderliche Optionen
- **`id:`**
  Die ID des Nutzers, der entbannt werden soll.

### Andere Optionen
- **`reason:`** [Optional]
  Der Grund für die Entbannung.

### Beschreibung
Hebt die Sperre eines Nutzers auf.

### Berechtigungen
