# 🎭❌ | Rolle entfernen

Befehl: **`/roles remove`**

Optionen:
- **`user:`** Der Nutzer, dem die Rolle entzogen werden soll.
- **`role:`** Die Rolle, die dem Nutzer abgenommen werden soll.
- **`duration:`** [Optional]  
  Die Dauer, wie lange die Rolle entfernt bleiben soll (Angabe in z. B. "5h", "10m").

<u>Beschreibung:</u>  
 Entfernt eine Rolle von einem Nutzer permanent oder für einen festgelegten Zeitraum.

<u>Berechtigungen</u>  
 Administratoren / Moderatoren
