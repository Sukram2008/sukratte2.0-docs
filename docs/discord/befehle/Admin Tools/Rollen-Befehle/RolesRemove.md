# ➖🎭 | Rolle entfernen

Befehl: **`/roles remove`**

Optionen:
- **`user:`**
  Der Nutzer, dem eine Rolle entzogen werden soll.
- **`role:`**
  Die Rolle, die entfernt werden soll.

<u>Beschreibung:</u>  
  Entfernt die ausgewählte Rolle von dem angegebenen Mitglied.

<u>Berechtigungen</u>  
  Administratoren (Rollen mit "Rollen verwalten")
