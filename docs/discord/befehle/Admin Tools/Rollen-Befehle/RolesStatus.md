# 📊🎭 | Rollen-Status

Befehl: **`/roles status`**

Optionen:
- **`user:`** [Optional]  
  Nutzer von welchem du temporäre Rollen sehen willst.

<u>Beschreibung:</u>  
  Zeigt eine Übersicht der Rollen eines Nutzers an. Laut Screenshot ist das User-Feld optional, da die Beschreibung darüber angezeigt wird.

<u>Berechtigungen</u>  
  Administratoren
