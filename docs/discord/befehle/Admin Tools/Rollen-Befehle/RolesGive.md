# ➕🎭 | Rolle vergeben

Befehl: **`/roles give`**

Optionen:
- **`user:`**
  Der Nutzer, der eine Rolle erhalten soll.
- **`role:`**
  Die Rolle, die zugewiesen werden soll.

<u>Beschreibung:</u>  
  Vergibt die ausgewählte Rolle an das angegebene Mitglied.

<u>Berechtigungen</u>  
  Administratoren (Rollen mit "Rollen verwalten")
