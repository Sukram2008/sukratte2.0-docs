# 📥🎨 | Emote hinzufügen

Befehl: **`/stealemote`**

Optionen:
- **`emote:`**
  Der Emote, der permanent geliehen werden soll.

<u>Beschreibung:</u>  
  Leiht einen Emote von einem anderen Server permanent aus und fügt ihn deinem Server hinzu. Das Feld ist eine Pflichtangabe.

<u>Berechtigungen</u>  
  Administratoren
