# 🔄🏷️ | Rollenposition ändern

Befehl: **`/admin moverole`**

Optionen:
- **`role:`** Die Rolle, welche bearbeitet werden soll.
- **`new-position:`** Die Position, auf welche die Rolle verschoben werden soll.

<u>Beschreibung:</u>  
 Ermöglicht das Verschieben von Rollen innerhalb der Server-Hierarchie direkt per Command.

<u>Berechtigungen</u>  
 Administratoren
