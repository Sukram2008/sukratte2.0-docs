# 📂📌 | Kategorie zuweisen

Befehl: **`/admin setcategory`**

Optionen:
- **`channel:`**
  Der Kanal, dessen Kategorie geändert werden soll.
- **`category:`**
  Die Ziel-Kategorie, in die der Kanal verschoben werden soll.

<u>Beschreibung:</u>  
  Verschiebt einen Kanal in die angegebene Kategorie.
