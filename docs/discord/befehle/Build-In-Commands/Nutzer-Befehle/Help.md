# 📖❓ | Hilfe-Menü anzeigen

Befehl: **`/help`**

### Beschreibung
Öffnet das interaktive Hilfe-Menü des Bots. 
- Du erhältst eine Liste aller Module, auf die du Zugriff hast.
- Über das Dropdown-Menü kannst du detaillierte Informationen zu einzelnen Befehlskategorien abrufen.
-# Dies ist ein Build-In-Command
