# 🛡️⚪ | Whitelist anzeigen

Befehl: **`/ping-protection list whitelisted`**

### Beschreibung
Zeigt eine Liste aller Nutzer an, die die Erlaubnis haben, geschützte Nutzer trotz aktivem Ping-Schutz zu markieren.
