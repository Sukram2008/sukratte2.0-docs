# 🛡️🔒 | Geschützte Nutzer anzeigen

Befehl: **`/ping-protection list protected`**

### Beschreibung
Gibt eine Übersicht über alle Nutzer aus, für die der Ping-Schutz aktuell aktiviert ist.

### Berechtigungen
