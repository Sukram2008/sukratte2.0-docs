# 🌦️💰 | Aktiviere Nachrichten-Drops

Befehl: **`/economy msg_drop_msg enable`**

### Beschreibung
Aktiviert die zufälligen Gelddrops im Chat. Bei Aktivierung können Nutzer beim Schreiben von Nachrichten zufällig Geld finden.

### Berechtigungen
