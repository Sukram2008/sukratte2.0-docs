# ➕💰 | Geld hinzufügen

Befehl: **`/economy add`**

### Erforderliche Optionen
- **`user:`**
  Der Nutzer, der das Geld erhalten soll.
- **`amount:`**
  Der Betrag, der hinzugefügt werden soll.

### Beschreibung
Mit diesem Befehl kannst du einem spezifischen Nutzer einen beliebigen Geldbetrag gutschreiben.

:::warning Achtung: Unfairness-Gefahr
Das manuelle Hinzufügen von Geld verfälscht die serverweite Wirtschaft. Es wirkt gegenüber Spielern, die sich ihr Geld ehrlich erarbeiten, unfair. Nutze diesen Befehl nur für offizielle Belohnungen.
:::
