# 🗑️❌ | Item löschen

Befehl: **`/shop delete`**

### Optionen
- **`item-name:`**
  Der Name des zu löschenden Items.
- **`item-id:`**
  Die ID des zu löschenden Items.

:::info Tipp: Sicherer Weg
Nutze am besten die **`item-id`**, um sicherzustellen, dass das korrekte Item dauerhaft entfernt wird.
:::

### Beschreibung
Entfernt ein Item unwiderruflich aus dem Shop-Sortiment. Bereits gekaufte Rollen bleiben bei den Nutzern bestehen.
