# 📊🔍 | Spielstatus

Befehl: **`/guess-the-number status`**

### Beschreibung
Zeigt dir an, ob in diesem Kanal gerade ein aktives Errate-die-Zahl-Spiel läuft und welcher Zahlenbereich (Min/Max) aktuell gültig ist.

### Berechtigungen
