# 🎲🆕 | Spiel erstellen

Befehl: **`/guess-the-number create`**

### Erforderliche Optionen
- **`min`**
  Der niedrigste Wert, den Nutzer raten können
- **`max`**
  Der höchste Wert, den Nutzer raten können

### Andere Optionen
- **`number`** [Optional]
  Die exakte Zahl, welche die Nutzer erraten sollen (ansonsten wählt der Bot eine Zufallszahl)

### Beschreibung
Mit diesem Befehl startest du ein neues Gewinnspiel in einem Kanal. Du legst den Zahlenbereich fest, in dem sich die gesuchte Zahl befinden muss.
