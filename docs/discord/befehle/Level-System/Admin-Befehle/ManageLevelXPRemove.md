# ➖✨ | XP entfernen

Befehl: **`/manage-levels edit-xp remove`**

### Erforderliche Optionen
- **`user:`**
  Benutzer, dem XP abgezogen werden sollen.
- **`value:`**
  Anzahl der XP, die entfernt werden sollen.

### Beschreibung
Verringert die Erfahrungspunkte eines Benutzers um den angegebenen Wert.

### Berechtigungen
