# 👤📊 | Profil

Befehl: **`/profile`**

### Andere Optionen
- **`user:`** [Optional]
  Der Nutzer, dessen Profil du einsehen möchtest (Standard: Du).

### Beschreibung
Zeigt dein aktuelles Level, deine XP und deinen Rang auf diesem Server an. Wenn du einen anderen Nutzer angibst, siehst du dessen Profilkarte.
