# 📋👥 | Liste der Nutzer mit Zugriff auf den temporären Kanal

Befehl: **`/temp-channel list-users`**

### Beschreibung
Gibt eine Übersicht aller Mitglieder aus, die aktuell manuell zu deinem temporären Kanal hinzugefügt wurden.

### Berechtigungen
