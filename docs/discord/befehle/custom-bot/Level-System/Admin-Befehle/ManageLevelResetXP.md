# 🔄✨ | XP zurücksetzen

Befehl: **`/manage-levels reset-xp`**

### Andere Optionen
- **`user:`** [Optional]
  Benutzer, dessen XP zurückgesetzt werden sollen (Standard: Du).
- **`confirm:`** [Optional]
  Bestätige die Aktion, um die XP endgültig zu löschen.

### Beschreibung
Mit diesem Befehl werden alle gesammelten Erfahrungspunkte eines Benutzers gelöscht. Wenn kein spezieller Benutzer ausgewählt wird, setzt der Befehl deine eigenen XP zurück.
