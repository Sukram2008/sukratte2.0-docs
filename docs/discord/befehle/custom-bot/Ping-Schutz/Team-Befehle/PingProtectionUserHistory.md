# 🛡️📜 | Ping-Verlauf

Befehl: **`/ping-protection user history`**

### Erforderliche Optionen
- **`user:`**
  Der Nutzer, dessen Ping-Historie überprüft werden soll.

### Beschreibung
Listet die letzten Male auf, an denen der gewählte Nutzer markiert wurde, während sein Ping-Schutz aktiv war.
