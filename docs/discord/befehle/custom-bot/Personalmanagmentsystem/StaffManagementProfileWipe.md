# ⚠️🎭 | Staff-Profil löschen (Wipe)

Befehl: **`/staff-management profile wipe`**

:::danger Warnung
Dieser Vorgang ist **unwiderruflich**. Alle Statistiken, Einträge und historischen Daten des Nutzers werden dauerhaft gelöscht.
:::

### Andere Optionen
* **`user:`** Der Nutzer, dessen Profildaten gelöscht werden sollen.
