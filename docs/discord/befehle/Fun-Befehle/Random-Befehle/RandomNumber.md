# 🔢🎲 | Zufallszahl

Befehl: **`/random number`**

### Andere Optionen
- **`min:`** [Optional]
  Die kleinstmögliche Zahl.
- **`max:`** [Optional]
  Die größtmögliche Zahl.

### Beschreibung
Generiert eine zufällige Zahl zwischen dem angegebenen Minimum und Maximum.

### Berechtigungen
