# 🖐️💥 | Slap

Befehl: **`/slap`**

### Erforderliche Optionen
- **`user:`**
  Der Nutzer, dem du eine Schelle geben möchtest.

### Beschreibung
Verpasse einem anderen Mitglied eine virtuelle Schelle (Random GIF).
