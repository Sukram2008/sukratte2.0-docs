# 🌦️💰 | Deaktiviere Nachrichten-Drops

Befehl: **`/economy msg_drop_msg disable`**

### Beschreibung
Deaktiviert die zufälligen Gelddrops im Chat. Nutzer können dann kein Geld mehr durch Nachrichten finden.

### Berechtigungen
